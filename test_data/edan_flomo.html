#读书笔记

臭鼬工厂（Skunk Works）在飞机制造这一高安全要求的行业中，通过独特的组织管理模式，成功实现了“少而精的团队+简单灵活的流程”与“极致安全”的平衡。其核心逻辑在于通过**高度赋权、精英化协作、责任共担**和**创新导向的文化**，而非依赖繁琐的流程机制。以下是具体实现方式及其内在逻辑：

---

### 一、**少而精的团队：精英化协作与责任共担**

1. **严格筛选人才**

臭鼬工厂的团队规模极小（如最初仅23名工程师），但成员均为领域内顶尖人才。他们不仅技术能力卓越，且具备跨学科协作能力，能够独立承担任务并快速解决问题。

- 例如，在XP-80喷气式战斗机的研发中，团队仅用143天完成任务，比合同提前37天交付，核心在于成员的高效协作与个人能力。

2. **责任全流程绑定**

凯利·约翰逊强调设计者和制造者必须全程参与测试，并对产品负全责。例如，臭鼬工厂的工程师需要对自己设计的部件负责到底，若出现问题需立即修复，这种责任共担机制减少了流程中的监管依赖。

3. **扁平化决策与高度赋权**

项目经理拥有绝对控制权，可直接向公司高层汇报，避免了官僚层级导致的效率损耗。团队成员被赋予充分自主权，能够快速决策并执行，无需层层审批。

---

#领域/AIGC

有时，语言模型需要进行详细的逐步推理才能回答特定问题。如果过于匆忙得出结论，很可能在推理链中出现错误。因此，我们可以通过“思维链推理”（Chain of Thought Reasoning）的策略，在查询中明确要求语言模型先提供一系列相关推理步骤，进行深度思考，然后再给出最终答案，这更接近人类解题的思维过程。

主要是因为链式提示它具有以下优点:

  1. 分解复杂度，每个 Prompt 仅处理一个具体子任务，避免过于宽泛的要求，提高成功率。这类似于分阶段烹饪，而不是试图一次完成全部。

  2. 降低计算成本。过长的 Prompt 使用更多 tokens ，增加成本。拆分 Prompt 可以避免不必要的计算。

  3. 更容易测试和调试。可以逐步分析每个环节的性能。

  4. 融入外部工具。不同 Prompt 可以调用 API 、数据库等外部资源。

  5. 更灵活的工作流程。根据不同情况可以进行不同操作。

综上，链式提示通过将复杂任务进行科学拆分，实现了更高效、可靠的提示设计。它使语言模型集中处理单一子任务，减少认知负荷，同时保留了多步骤任务的能力。随着经验增长，开发者可以逐渐掌握运用链式提示的精髓。

第一次工业革命是机械化革命，诞生于18世纪晚期，瓦特改良蒸汽机后，1784年问世的珍妮纺织机实现了生产的机械化，从而催生了车间、工厂这种新型生产组织模式，大量农民与手工业者成为产业工人，机械化革命成就了英国日不落帝国的传奇，也促使人类从农耕文明走向了工业文明。

第二次工业革命是电气化革命，标志事件是1870年，美国辛辛那提屠宰场的自动化生产线建成，企业能够大批量、流水线式生产，该时期以福特汽车为代表，工业进入了大批量的生产阶段。

第三次工业革命是信息化革命，是20世纪70年代开始的，核心技术就是PLC（可编程逻辑控制器）以及后来的互联网等信息化支撑下的自动化得到进一步发展。进入信息化革命后，人们可以从繁复的劳动中得到初步解放，从而将更多精力投入到创造性工作当中。从这以后，技术进步呈加速状态，新兴技术转化为产品的周期越来越短。

第四次工业革命就是现在进行中的工业4.0，通过赛博物理系统（Cyber-Physical Systems, CPS）实现物理世界在数字世界的精确映射，打造“数字孪生”，实现物理实体与数字虚体之间的互联、互通、互操作，最终将智能机器、存储系统和生产设施融入整个生产系统中，人、机、料等能够相互独立地自动交换信息、触发动作和自主控制，实现一种智能的、高效的、个性化的、自组织的生产与服务方式，推动制造业向智能化转型。

作为工业4.0的一个核心组成，智能工厂将渗透到公司间的价值网络中，并最终促使数字世界和现实的完美结合。”

在德国人的愿景中，工业4.0是一种基于自动化、数字化、网络化的智能化生产模式。工厂内，人、机、料自主协同，自组织、高效运转；工厂外，通过端到端集成、横向集成，实现价值链的共享、协作，效率、成本、质量、敏捷性都得到了质的飞跃。

每日问题：为什么提出了工业4.0？

在本轮智能制造热潮中，除了德国的“工业4.0”、美国的“先进制造业国家战略计划”、中国的“中国制造2025”、日本的“机器人新战略”等大家耳熟能详的国家级战略以外，英国、法国、韩国、印度、俄罗斯等众多国家也推出了一系列战略，虽然名称、时间点、侧重点各不同，但智能制造是这些国家战略的共同核心，各国都期望借助于自动化、数字化、网络化、智能化手段，减少对人的依赖，实现各自国家向高质、高效、高端、绿色、高竞争力方向发展。大家面临的问题主要有：

  • 难以提升的经济增长：近些年来，世界经济陷入低迷，老牌工业国家一蹶不振，被寄予厚望的新兴工业国家发展速度也开始放缓，全球经济很难找到有效的快速发展途径；gdp增长一路下跌

  • 难以逆转的老龄化趋势：困扰当今经济的另一大因素就是全球老龄化趋势进一步凸显。我们正感受到企业招人难、用工难，中国制造业的人口红利在逐渐丧失，实际上，这才是刚刚开始，很多国家已经被这个问题困扰很长时间了。严峻的老龄化与低生育率已经导致了人口结构的重大变化，在客观上倒逼着社会的变革，包括制造业新战略的制订与推出，这也是上述国家制订智能制造战略的另一个重要原因。

而在cpc时代，ecmp=出价*ctr*1000*曝光量。算法模型只需要预测ctr。然后从参与竞价的选手中选出ctr最高的广告即可。

基因里的DNA决定了细胞中RNA合成（转录），而RNA决定了蛋白质的合成（翻译）。

不同人身体中合成出来的蛋白质略有不同，有些特质和天分就会显示出细微的差异。

最著名的就是语言基因FOXP2的发现。

为什么只有人可以获得如此强大的语言能力，就连近亲大猩猩也做不到？

科学家研究了一个语言障碍家族，发现他们在FOXP2基因近27万个DNA碱基中出现了一个碱基的错误（碱基g变成了a），于是这些人产生的蛋白质分子中本来该是精氨酸的，却变成了组氨酸，导致在大脑中语言相关区域的神经元比正常人要少，最后造成了语言的障碍。

#读书笔记/文明之光

抗生素，是世界上最神奇的药物，可以一瞬间治愈各种外伤感染，肺结核，性病，霍乱等；

抗生素的发明故事要从青霉素谈起；

  1. 第一个发现青霉素的是英国的医生弗莱明，1928年莱明无意中发现了一类“霉”消灭了容器里的葡萄球菌；但是由于他没有化学知识，搞不清原理，所以虽然后续花了十年时间研究，但也没有稳定的实验结果，不过留下了一学术论文；

  2. 紧接着真正推动青霉素被药品化的是牛津大学拉德克利夫医院的弗洛里教授，1938年他成功从青霉菌中提炼出了青霉素；但是其浓度和提炼的量太低（全年提炼的量，才够治疗一两个病人）。后续弗洛里通过其号召力，做了大规模的社会组织，让多家制药公司、政府部门、大学科研室都参与到青霉素的量产研究中；再加上二战特殊时期对青霉素的强诉求，以及提供了大量的临床实验机会，所以成药效率非常快；

  3. 最终英国科学家亚伯拉罕在1943年，终于弄清楚了本质原因：青霉素之所以能够杀死病菌是因为青霉烷能使疾病的细胞壁的合成发生障碍，导致病菌溶解而死，但人和动物的细胞则没有细胞壁，所以不会有影响；

1945年，诺贝尔医学奖也颁发给了以上三位大神；

#读书笔记/文明之光

垄断企业是第二次工业革命带来的经济迅猛发展的附副产物；

  • 垄断出现在当时迅猛发现的一些行业（铁路，石油等）

  • 动机起初是希望稳定市场混乱的价格体系；

  • 垄断者最终构建出了“信托”的容器，来管理所有 联合的公司，这样避免了自然人做管理者的各种法律限制；

其实行业垄断期初带来的是产品的成本下降，因为只有足够高的生产效率，低的成本低产品才能最后通吃市场；但是长期上，危害大于益处；

  • 破坏了社会的公平性，新的企业根本无法进入对应的领域，这长远上会遏制整个社会的活力和高速发展；

  • 虽然洛克菲勒的石油王国客观上把石油价格降的很低，而且作风也自律节省，但是不是每一个 都能像他一样，企业垄断市场后可以直接控制市场，提高价格损害人民经济利益；

  • 造成当时的美国政府腐败，很多大企业直接买通政府高层，建立有利于垄断企业的市场法律规则；政客问了升职，也需要寻求大企业帮助

老罗斯福之所以位列美国拉什莫尔山的总统雕像之一，主要是因为他在任期间发起了发垄断运动，推动颁发了《谢尔曼反垄断法》，亲手把北方证劵（实际是铁路垄断企业），标准石油公司等40家企业直接拆分。维护了美国社会的公平与自由。

我个人认为，在行业迅猛发展的阶段，可以不要因为垄断而过多的干预，这样可以最快提高服务的质量，降低成本。但是当一切成熟进入发展期时，可以展开反垄断进一步干预以，以保证更多有创新的企业进入；
